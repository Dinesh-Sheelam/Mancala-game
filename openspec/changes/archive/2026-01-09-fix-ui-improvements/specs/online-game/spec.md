## ADDED Requirements

### Requirement: Online Game Navigation
The Create Room interface SHALL include a "Back to Game Selection" button matching the Join Room interface for consistent navigation.

#### Scenario: User navigates back from Create Room
- **WHEN** the user views the Create Room interface
- **THEN** a "Back to Game Selection" button is displayed at the bottom
- **AND** the button matches the styling of the Join Room interface
- **AND** clicking the button navigates to the Game Selection page
